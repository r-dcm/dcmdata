# dcmdata: Data Sets for Diagnostic Classification Modeling

Access data sets for demonstrating or testing diagnostic classification
models. Simulated data sets can be used to compare estimated model
output to true data-generating values. Real data sets can be used to
demonstrate real-world applications of diagnostic models.
