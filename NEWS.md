# dcmdata 0.2.0

* Added new data from the Pathways for Instructionally Embedded Assessment (PIE)
  project: `pie_ft_data`, `pie_ft_qmatrix`, `pie_pilot_data`,
  `pie_pilot_qmatrix`.

* Added new data sets from the [Item Response Warehouse](https://itemresponsewarehouse.org/).
  * Fraction subtraction: `fraction_data`, `fraction_qmatrix`.
  * MCMI-III: `mcmi_data`, `mcmi_qmatrix`.
  * ROAR-PA: `roarpa_data`, `roarpa_qmatrix`.
  * TIMSS (2003): `timss03_data`, `timss03_qmatrix`.
  * TIMSS (2007): `timss07_data`, `timss07_skill_qmatrix`, `timss07_topic_qmatrix`, `timss07_domain_qmatrix`.

* Added a `NEWS.md` file to track changes to the package.
